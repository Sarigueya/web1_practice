# 🌐 Web1 Practice — Prácticas de Desarrollo Frontend

Colección de ejercicios prácticos de desarrollo web frontend enfocados en la aplicación de **Bootstrap**, **CSS** y **JavaScript** con validación de formularios mediante expresiones regulares.

---

## 📋 Contenido

- Maquetación y diseño responsivo con **Bootstrap**
- Estilos personalizados con **CSS**
- Validación de formularios con **JavaScript** y expresiones regulares

## 🚀 Cómo ver el proyecto

No requiere instalación. Simplemente clona el repositorio y abre el archivo `index.html` en tu navegador:

```bash
git clone https://github.com/Sarigueya/web1_practice.git
cd web1_practice
# Abrir index.html en el navegador
```
